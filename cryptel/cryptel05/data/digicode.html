1. DESCRIPTION

Les PTC 21 SN, PTC 21, PTC 22, PTC24 sont des portiers téléphoniques detinés à être connectés sur autocommutateurs privés ; ils possèdent 1 récepteur de télécommande de Gâche, 1 sortie Information "Prise de ligne" (sur relais) pour commande d'éclairage ou de caméra. En option : l'info "Prise de ligne" devient un 2eme récepteur pour Gâche

PTC 21 SN : Appel par prise de ligne (numérotation par l'autocommutateur)
PTC 21 : 1 numéro pré-enregistré
PTC 22 : 2 numéro pré-enregistrés
PTC 24 : 4 numéro pré-enregistrés

 

2. RECOMMANDATIONS

Il est conseillé d'utiliser un cable blindé entre le boitier et l'autocommutateur, le blindage étant relié à la terre d'un seul côté. Le boitier doit être relié à la terre

 

3. INSTALLATION

Les portiers sont livrés avec un capot de protection, permettant l'encastrement dans les parois minces (1 à 4 mm). Des boîtiers pour encastrement en maçonnerie et pour montage en saillie sont disponibles, pour ces 2 derniers modèles poser impérativement le joint assurant l'étanchéité de la façade.

 

4. RACCORDEMENT (voir annexe 1)

  1. Ligne téléphonique : relier au bornier CO5 en L1 L2
  2. Mise à la terre : relier la borne de CO5 à la terre
  3. Eclairage façade (option) : relier l'alimentation 12V continu au connecteur prévu à cet effet en respectant la polarité, caractéristiques de l'alimentation 12V, 0.1A
  4. Raccordement Gâche : les sorties relais Gâche sont constituées de relais électromécaniques. Pouvoir de coupure 40VA/48 volts maximum. Chaque sortie est équipée d'un contact repos et d'un contact travail

 

Nb:
- Raccordement (voir annexe 1)
- Gâche N1, sorties relais T1, C1, R1
- Info "prise de ligne", sorties relais T2, C2, R2 ou Gâche N2 (option)

 

5. INITIALISATION (voir annexe 1)

A sa sortie d'usine, le portier est livré pour fonctionnement sur autocommutateur privé, cavalier JP 2 sur position 2. En cas de fonctionnement sur P.T.T mettre JP 2 sur position 1.

Décrocher automatique
Pour permettre le téléchargement, l'appareil est livré sur la position décrocher automatique JP 1 sur position 2

Longueur de ligne
Le portier est configuré en ligne courte CO17 en position 1 et CO18 en position 2 (< à 500 m).
Possibilité de ligne longue (> à 500 m) CO17 position 2 et CO18 position 1.

Réglage du volume d'écoute
Ajuster à l'aide du potentiomètre P 1.

 

6. PROGRAMMATION (voir annexe 2 et 3)

La programmation concerne :

  1. Le type de numérotation décimale ou DTMF

  2. Le raccrocher par le bouton d'appel ou non

  3. La durée maximale de communication (1 à 9 minutes) ou illimitée

  4. Le ou les numéros préprogrammés (PTC 21, PTC 22, PTC 24)

  5. L'autorisation de commande Gâche N°1

  6. Le temps d'activation du relais Gâche N°1

  7. L'autorisation de Gâche N°2 (option

  8. Le temps d'activation du relais Gâche N°2 (option)

  9. Le code secret (d'accès à la programmation) à sa sortie d'usine est : 0.0.0.0 (4 chiffres)

  10. Le code d'identification (5 chiffres)

  11. Le ou les codes de Gâche N°1

  12. Le ou les codes de Gâche N°2 (option)

Voir annexe 2 : menu de programmation par clavier local
Voir annexe 3 : menu de programmation par Q23 (DTMF)
Pour le téléchargement voir notice de programmation des portiers Série 20 par Minitel

 

Exemples de programmation de numéro pré-enregistrés

PTC 21 (1 numéro d'appel pré-enregistré)

  1. Prise de ligne : appui sur le bouton de la face avant

  2. Programmation : appui sur la touche "P" du clavier = 2 bips

  3. Code : 2.1 touches "2", "1" = 2 bips

  4. Numéro d'appel : (exemple 2810, touches "2", "8", "1", "0")

  5. Validation : "AP" = 3 bips

PTC 22 (2 numéros d'appel pré-enregistrés)

  1. Prise de ligne : idem PTC 21

  2. Programmation : touche "P"
    exemple : 1er numéro = 2810, appel intèrieur. 2eme numéro = appel extèrieur (0 = accès autocom) 43.67.93.77

  3. Programmation des numéros de téléphone
    1er numéro code 21 = touches "2", "1" = 2 bips "2", "8", "1", "0" "AP" = 3 bips
    2eme numéro code 22 = touches "2", "2" = 2 bips "0", A ou "B", "4", "3", "6", "7", "9", "3", "7", "7", "AP" = 3 bips
    A = attente de tonalité
    B = pause de 2 secondes

  4. Touche P = fin de programmation

 

7. Utilisation

PTC 21 SN
Appel au décrocher par appui sur le bouton, l'autocommutateur achemine l'appel vers le poste préselectionné.

PTC 21, 22, 24
Appel par appui sur un bouton, l'appareil compose automatiquement le numéro pré-enregistré correspondant.

Télécommande de gâche
Les PTC 21 SN, 21, 22, 24 possèdent : 1 relais de télécommande de gâche et 1 relais d'info "prise de ligne", ou 2 relais de gâche (en option). Pendant la communication le correspondant peut depuis son poste (DTMF), commander la ou les gâches en composant le code sur son clavier suivi du "#", 2 bips confirment le bon code.
NB: Le relais de prise de ligne donne un contact pendant toute les communication permettant de commander un éclairage ou un caméra
Pour arrêter la communication il suffit au correspondant de raccrocher, le portier détectera la tonalité d'occupation et raccrochera. Si le correspondant fait suivre le code du "*" le raccrocher est immédiat.
