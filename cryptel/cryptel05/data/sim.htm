DESIMLOCKAGE de TELEPHONE PORTABLE


POURQUOI LES TELEPHONES SONT-ILS VEROUILLES ?

Lorsque vous achetez un portable bon marché, l'opérateur désire que vous utilisiez SON réseau en contre-partie de la réduction que vous avez obtenu. L'opérateur est en mesure de le faire grace a un contrat signé, OU en verouillant le téléphone pour que vous ne puissiez utiliser un opérateur différent avec ce téléphone.

CE QUI EST VERROUILLÉ:

Un téléphone peut être verouillé, mais pas la carte SIM! La SIM elle-meme n' est pas verouillée: vous pouvez l' utiliser dans n'importe quel téléphone portable compatible.

COMMENT SAVOIR SI LE TÉLÉPHONE EST VERROUILLÉ:

Seulement les téléphones achetés dans un " pack " (du style mobicarte ou OLA), ainsi qu'une SIM, peuvent être verrouillés. Si le téléphone est verrouillé, c'est toujours indiqué sur l' emballage (raisons légales). Si vous devez signer un contrat avant qu'on vous laisse emporter le téléphone, il n' est peut-etre pas verrouillé. Mais si vous n'avez pas du tout à vous identifier, il est certainement verouillé.

Un téléphone est susceptible d'être verrouillé si:

    - il est vraiment bon marché;
    - vous n'avez pas eu à donner votre nom lors de l'achat
    (exemple: les téléphones à cartes prépayées)

Un téléphone est susceptible de ne pas être verrouillé si:

    - il est vraiment vieux (Nokia 2110 etc ..);
    - vous avez signé un contrat d'abonnement d'au minimum 1 an;
    - vous achetez un portable perdu sans carte SIM (evidemment..);

Mais vous pouvez être sur seulement en essayant avec une SIM différente. Aucun endommagement n'aura lieu.

LES DIFFERENTS TYPES DE VERROUILLAGES

    - Verrouillage du portable pour une SIM
    - Verrouillage du portable pour un opérateur (network lock)
    - Verrouillage du portable avec un code (product lock)

Si votre téléphone est verrouillé pour une SIM, vous ne pouvez utiliser aucune autre SIM que celle que vous avez: une SIM cassée rend le téléphone inutilisable.

Si votre téléphone est verrouillé pour un opérateur, vous pouvez utiliser n'importe quelle SIM de cet opérateur dans votre téléphone. Vous pouvez commuter entre un abonnement et une carte prépayée comme vous le souhaitez, mais pas pour un opérateur différent.

Si votre téléphone est verrouillé avec un code, le téléphone contrôle un " mot de passe " spécial (GID) sur la SIM qui doit concorder avec le téléphone. Les fournisseurs utilisent souvent différents mots de passe pour l'abonnement et les cartes prépayées.

Note: un téléphone peut avoir plus d'un verrouillage actif simultanément!

POURQUOI DEVERROUILLER

Par exemple:

    - pour utiliser un autre réseau de meilleure qualité;
    - pour utiliser votre portable en vacances;
    - pour commuter entre un abonnement et une carte prépayée;

Note: parfois cela fonctionne également sans déverrouillage.

RISQUES DU DEVERROUILLAGE

Il y a des risques en déverrouillant votre téléphone:

    - la garantie;
    - les modifications (Ericsson);
    - la casse;

Si votre téléphone doit retourner à la société pour n'importe quelle réparation, elle peut se rendre compte du déverrouillage que vous avez effectué. Quelques téléphones Ericsson sont protégés contre le déverrouillage: après avoir essayé de les déverrouiller, ils peuvent afficher l'expression " tampered" et sont ensuite totalement bloqués. Vous devez alors les rapporter au magasin.

COMMENT DÉVERROUILLER

Il existe plusieurs méthodes pour déverrouiller la serrure de la SIM:

    - Méthode simple: par le biais de votre opérateur/constructeur
    - Méthode complexe: faites-le vous même ou faites-le faire
    par quelqu'un d'autre.

COMMENT DÉVERROUILLER: méthode simple

Demandez simplement à votre fournisseur.

Parfois c'est gratuit, dans d'autres cas vous devez payer . Ils peuvent vous envoyer un message SMS pour déverrouiller le périphérique (recu bien entendu avec l'ancienne SIM), en général ils vous envoient un code que vous devez entrer dans votre portable pour commuter avec une nouvelle SIM.

COMMENT DEVERROUILLER: méthode complexe

    - Charger un software pirate qui permette de supprimer le controle
    - Charge une version differente de software dans le portable
    - Changer le data du portable comme si le verrouillage était officielement enlevé
    - Changer le data du portable pour l'utilisation d'une autre SIM

Vous avez besoin de câbles et de logiciels spéciaux pour faire ces modifications. Notez que changer le logiciel d'un portable est ILLÉGAL, le constructeur ayant un droit sur ce logiciel. D'autres méthodes de déverrouillage ne sont pas illégales (si le logiciel que vous utilisez n'est pas illégal).

Les magasins ne vendent en général pas de câbles ou de logiciels, mais vous pouvez les acheter sur le net. Ce genre de matériel est plus difficile à trouver pour certaines marques comme Nokia, et vous devez plutôt chercher d'autres épellations telles que "n0kia" dans les moteurs de recherche pour en trouver, car Nokia les chasse du marché. Pour les Nokia, le logiciel " Wintesla " est utilisé pour tester, reparer, déverrouiller, etc.. tous les modèles de la marque. Wintesla possède une librairie s'accordant aux différents modèles de Nokia. (NHE-1,3,4, NHE-5,6, NHE-8,9, NHK8,9, NSE-1,3 (pour nokia 5110 et 6110), NSK-1,3, NSB-3 et NSM-1. )

PUIS-JE UTILISER LE CODE D'AUTRES PERSONNES?

Non, le code de déverrouillage que vous avez eu par votre opérateur est différent pour chaque portable!

QU'EST-CE QUE LE IMEI, EN AI-JE BESOIN?

Le numéro d'IMEI est le numéro de série électronique de votre téléphone. Tous les GSM l'affichent une fois tapé *#06#. Ca sert à l'administration pour les opérateurs. Il est inutile pour le déverrouillage.

EST-CE QUE J'UTILISE la BROCHE, le PUK, le PIN2 ou le PUK2?

Non, ce sont des codes totalement différents et n'ont rien à voir avec le verrouillage de la carte SIM.

SIM-CLOCK = SIM-LOCK?

Non, l'horloge (CLOCK) est un terme électrique et n'a rien à voir avec le verrouillage.

UN VERROUILLAGE SIM EST-IL LEGAL?

Parfois oui, parfois non. En Belgique on ne permet pas aux opérateurs de verrouiller les SIM. Aux pays-bas, ils ont ce droit, mais doivent retirer le lock sur demande contre de l'argent.

RETIRER UN SIM LOCK EST-IL LEGAL?

Si vous ne signiez pas de contrat avec l'opérateur et que vous ne violez pas des règles de copyright, alors il est légal de retirer un verrouillage de SIM.

Pin number is counted with keyboard up, from left to right

1: AFMS, Out, Audio from mobile station (external speaker, very small signal 10 mV).

10: GND, Digital Ground and DC Return
11: TTMS/DTMS, In, Terminal Adaptor Equipment To Mobile Station/Data To Mobile Station (Serial input).
12: DCIO, In/Out, DC positive pole for phone battery charging and external accessory powering (Charger plus).
