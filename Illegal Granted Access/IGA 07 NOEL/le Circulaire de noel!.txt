Voici les 13 commandes a utiliser lorsque vous etes connecter[les plus connus]:


        CD      : Change Directory [Deplacer vous dans les repertoires]
        DIR     : Directory Listing [Afficher le contenue dun repertoire]
        RECV    : Downloader un fichier
        SEND    : Uploader un fichier
        BIN     : Definir le type de packet lors des uploads/downloads
        PWD     : Afficher le repertoire en cour
        QUIT    : Sortir du programme
        USER    : Reloggez-vous sur un autre user
        CLOSE   : Fermer le FTP distant
        HASH    : Mettez un marqueur de transfert
        MKDIR   : Creer repertoire
        RMDIR   : Effacer Repertoire
        DELETE  : Effacer Fichier.
        LS      : Listing with attributes


CD
--

C'est pareil comme en MS-DOS, si sur votre ftp vous avez un tree comme ca:


.
..
/pub/
/pub/tonefucker
welcome.txt
/etc


vous faite cd /pub/ pour allez dans le repertoire pub ou encore
cd /pub/tonefucker pour allez dans le repertoire /pub/tonefucker.. C'est
pas tres dure a retenir de se coté la. TOUT les fichiers et repertoires
cacher commence par un point(.) se que CuteFtp ne detecte pas :-)




DIR
--

Voir le contenu du repertoire, la liste des fichiers.
Pareil qu'en MSDOS finalement.



RECV Remotefile LocalFile
-------------------------

Admettons que dans /pub/ il y a miamiam.zip a Downloader, vous devez 
tapez par exemple:

RECV /pub/miammiam.zip D:\download\mm.zip

ici le D:\download\mm.zip est la destination sur votre systeme, ne
donner pas un nom deja existant , il va ecraser le fichier existant
sans vous le dire...



SEND Localfile RemoteFile
-------------------------

Le contraire de RECV, cest a utiliser quand VOUS envoyez des données
au serveur. Si par exemple vous avez d:\download\mm2.zip a envoyer sur
le serveur dans le repertoire /pub/miamiam2.zip vous avez a tapez:

SEND d:\download\mm2.zip /pub/miamiam2.zip


BIN
---

Il existe 2 types de transfert, le mode texte [par defaut]  ou   le mode
binaire. Si vous downloader des ZIP ou des trucs comme  ca   vous  devez
etre en binaire, si vous downloader des textes le  ascii    suffit,  une 
regle d'usage est de le laisser en binaire. Alors apres vous etes entrer
sur le FTP tapez toujours BIN 1 fois seulement.



PWD
---

Permet de voir dans quel repertoire on se situe, exemple:


Ftp> pwd
/yousuxx/alittle/bit/my/dear/



QUIT
----

L'essayer c'est l'adopter! :-)




USER
----

Vous re-entrez dans le serveur avec la bonne vieille sequence USER/LOGIN


CLOSE
-----

Vous fermez le lien et vous pourez en ouvrir un autre avec OPEN


HASH
----

Cest un petit add-on qui permet de voir ski arrive lors d'un download..
car d'habitude quand vous downloader/uploader (transfert) vous ne voyez
rien qu'un ecran noir et a la fin vous avez la vitesse moyenne etc...le
HASH MARK cest simplement un gadget qui place un # a chaque 2048 bytes.

Tapez HASH 1 fois pour lactiver, une 2-eme pour le desactiver



MKDIR
-----

Equivalenet de MD en MS-DOS, permet la creation dun repertoire :

ex : Ftp> Mkdir /pub/zoulou




RMDIR
-----

Efface le repertoire, equivalent rd em MS-DOS :

ex: Ftp> Rmdir /pub/zoulou


DELETE
------

Efface un fichier, equivalent de DEL en Ms-DOS :

ex: Ftp> delete /pub/zoulou/miam2.zip
