Personnaliser son windobe cheri


Vous bootez sous windowbe ? Autant que ce soit ""agréable "" ! Je vais vous expliquer ici comment faire de son petit
OS cherie, une oeuvre d'art ;-)

Nous voulons donc personnaliser l'interface graphique, il existe pour cela deux logiciels tres connus => windowsBlinds et
Style XP. Nous parlerons ici juste de style XP tout simplement par le fait que je le prèfere a windowsBlinds ;-) !
Telecharger donc style XP (payant(mais il existe des cracks...)), et installez le.
Nous allons etudier un peu ce que ce programme a dans le ventre.
Faites demarer>TGTSoft>Style XP

OPTION
------
Les options utiles => RUN AT STARTUP => cochez "use no resources" ; Je pense que vous aurez compris pourquoi !! non ????
Et bien pour utiliser le moins de resources systemes possible !!

EXPLORER BAR
------------
Grace a cette partie, vous pouvez personnaliser la barre de l'explorer.

THEME
-----
Alors la ca commence a devenir interessant ! Grace aux theme, vous pourrez personnaliser totalement votre
interfase graphique ! Il existe peu de themes par defaut... Je vous invite donc a aller sut themexp.org, vous y
trouverez une multitude de thème aussi beau les uns que les autres !
Alors vous avez telecharger un theme, c'est bien Zolie ! Mais comment l'appliquer ?
cliquez sur "add a new theme" et allez chercher le fichier telcharger !

Visual Style
------------
Dans le meme genre que les themes ! Je vous reporte donc un peu plus haut !

BACKGOUND
---------
Permet de changer le fond d'écran

LOGONS
------
Là aussi ca devient interessant ! Grace a cette partie vous allez permettre de personnalier l'écran des sessions !
Vous savez ! Des que vous lancez windows, et si vous l'avez configurer pour cela, vous avez le choix entre les session
que vous avez definis !
Alors, aussi tres peu par defaut, je vous re-reporte a themexp.org.

ICONS
-----
Encore tres interessant ! Permet de personaliser vos icones

BootScreen
----------
Alors là, je vous conseille de ne pas toucher à ça du tout ! Permet de personnaliser l'écran de chargement de windows
(j'ai essayé une fois... et je l'ai tres vite regrété !)

TRANSPARENCY/ROTATE
-------------------
Peu interessant

CURSORS
-------
Permet de presonnaliser vos curseurs !

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Maintenant passons a une autre choses => Symetrix ! Ce petit Freeware vous permettra d'avoir un sacré nombre d'info
su votre pc (l'heure(...), Les resources systemes utilisés...etc.). Et puis surtout, le skin du logiciel est personalisable !

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Il est venu le temps (des cathedrales ? des rires et de chants ?) de vous donnez deux trois petits truc pour couroner le
tout.
+ Vous programmez ? Créez un petit programme de bienvenue assortis a votre Visual style, qui vous donne la date, l'heure
et un texte paramètrable qui vous fasse part des choses dont vous voulez vous rapeller !

+ Lancez une petite musique de fond a chaque demarage du pc =>
créez un petit fichier batch suivant cette syntaxe :

@echo off
start /MIN LeCheminDeVotreFichierMP3>nul
exit

+ Classez vos icones sur votre bureau (et essayez d'en avoir le moins possible !)

+ Ne les disposez pas a la traditionelle, VOUS AVEZ DE L'ESPACE sur votre bureau ! Ne stoquez pas tout a gauche boudiou !
