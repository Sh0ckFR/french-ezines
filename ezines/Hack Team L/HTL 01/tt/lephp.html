3] LES VARIABLES

Les variables, voici un concept essenciel de la programmation. Quelque soit le language,
vous retrouverez les variables .
Alors qu'est-ce ? Et bien je vais vous faire la celebre analogie de la boite !
En fait une variable est une boite, dans laquel je peut mettre tout ce que je veux :
les lettres, des chiffres ...etc. Alors a quoi cela sert il ?

Et bien imaginez, que dans votre programme vous vouliez repeter 10 fois la phrase "bonjour m'sieur dame !" .
Au lieu de repeter 10 fois cette phrase, vous pouvez créer une variable qui s'apellera : "phrase"
et donc au lieu de repeter 10 fois la longue phrase, vous repeterez 10 fois "phrase" .

donc comment créer une variable?

$Nom_De_Ma_Variable = contenu;

exemple :

<?
$message="bonjour";
echo"$bonjour";
?>


Alors, pour les chaine de caractère, on met des "", pour des chiffre, pas besoin, exemple =>

Alors sachez que ce qui a de chiant dans les autre language de programmation, c'est que l'on doit indiquer qu'est-ce que
l'on vas mettre dans la variable (des chiffres ? Une chaine de caractères ? un chiffre a virgule ?) Mais en PHP, bande
de petits veinard, Pas besoin d'indiquer le type de donné que vas contenir la variable !
exemple d'une declaration de variable en VB6 =>

Dim Nom_De_Ma_Variable As Type

En C++ =>

Mais sachez qu'il est tout de meme possible d'indiquer le type de donné que l'on vas mettre dans la variable,
On peut FORCER le type. Mais avant tout, regardons un peu les types qu'il y a =>

Integer (ou int) => entier => exemple 25
float => nombre a virgule => 25.32
string => chaine de caract => "J'aime le PHP !!"
Boolean ou bool => bolléen => False ou True (vrai ou faux)

Maintenant, un exemple de Forcing de type :-D
$Nombre_de_testicules="2,5"; // normalement string car ""
