Les commandes mIRC sans script


1-DIVERS

Pour se mettre en away
/away message

Ecrire en couleur
[Ctrl-K]+(n° de la couleur) message

Souligner
[Ctrl-u] message

Ecrire en negatif
[Ctrl-r] message

Ecrire en gras
[Ctrl-B] message

Copier/Coller
[Ctrl-c] pour copier , puis [Ctrl-v] pour coller

2-ACTION AVEC NICKSERV

Aide
/msg nickserv help

Enregistrer un nick
/msg nickserv REGISTER mot_de_passe

S'identifier
/msg nickserv IDENTIFY mot_de_passe

Abandonner un nick
/msg nickserv drop

Infos sur un nick
/msg nickserv info nick

Changer de nick
/nick nouveau_nick

Changer de pass du nick
/msg nickserv set PASSWORD nouveau-pass

Deconnecter un utilisateur qui utilise le nick
??? /nickserv recover nick mot_de_passe

Tuer une connection ghost
??? /nickserv ghost nick mot_de_passe

Change le langage nickserv en francais
/msg nickserv set language 2

Statut du nick
/nickserv status nick

0 = nick non enregistre ou aucune personne a ce nick sur le serveur.
1 = la personne ayant ce nick ne s'est pas identifiee par nickserv.
2 = la personne ayant ce nick en est reconnue comme le proprietaire a partir de la liste d'acces du nickname, seulement.
3 = la personne ayant ce nick en est reconnue comme le proprietaire grace a l'identification aupres de nickserv.

3-Changenick

Aide
/nickserv help set changenick

Proteger le nick par un kill
/nickserv set changenick on ou off

Proteger le nick par un kill
/nickserv set changenick on ou off

Proteger le nick par un kill rapide (delai 60 sec ou 20 sec)
/nickserv set changenick quick on ou off

Proteger le nick par un kill immediat
/nickserv set changenick immed on ou off

Secure
/nickserv set secure on ou off

Infos Privees
/nickserv set private on ou off

4-Commandes sur un channel

Aide
/chanserv help

Enregistrer un channel
/msg chanserv register #channel mot_de_passe description_du_chan (phrase ou mot)

S'identifier en tant que fondateur d'un channel
/identify #channel mot_de_passe

Changer le pass
/chanserv set #channel PASSWORD nouveau-pass

Abandonner un channel
/msg chanserv drop #channel mot_de_passe

Kick les nicks sans acces (kick lis nicks non enregistres)
/msg chanserv set #channel restricted on ou off

Oppe uniquement les personnes ayant un acces de 5
/chanserv set #channel secureops on ou off (les personnes ne possédant pas d’access par chanserv ne pourront pas etre op)

Auto op les nick ou ????
??? /msg chanserv #chan autoop ????

Voice les nicks inconnus dans access list
/msg chanserv levels #chan set autovoice 0

Redonne la clef du chan
/msg chanserv getkey #channel (redonne la clef d'un +k)

Infos sur un channel
/chanserv info #channel

Message d'entree
/chanserv set #channel entrymsg Message_d'entree

5-Operations quand on est founder

Retirer tous les modes du chan
/chanserv clear #channel modes

Deopper tout les ops
/chanserv clear #channel ops

Devoicer tout un chan
/chanserv clear #channel voices

Retirer tous les bans
/chanserv clear #channel bans

Kicker tout le monde
/chanserv clear #channel users

Change le Founder
/chanserv set #chan FOUNDER nick

Successeur
/msg chanserv SET channel SUCCESSOR nick

Cacher le chan de la liste
/msg chanserv set #channel private on ou off

Envoie un notice quand OP/DEOP sont utilises.
??? /msg chanserv set #chan opnotice on ou off

Envoie un notice quand VOICE/DEVOICE sont utilises.
??? /msg chanserv set #chan VOICENOTICE on ou off

Bannir sans raison
/ban #chan nick

Bannir avec une raison
??? /ban #chan nick raison

6-Akick

Voir la liste des auto kick
/chanserv akick #chan view

Ajouter un nick a la liste
/chanserv akick #chan add nick

Effacer un nick de la liste
/chanserv akick #chan del nick

Effacer tous les nicks de la liste
/chanserv akick #chan clear

Inviter quelqu'un sur un channel
/invite nickname #channel

Opper
/chanserv op #channel nick

Deopper
/chanserv deop #channel nick

Voicer
/chanserv voice #channel nick

Devoicer
/chanserv devoice #channel nick

7-ACCES

Grades
1 = Rien
3 = auto-Voice
5 = Auto-Op
10 = Auto-Op protege
99 = Founder

Donner un acces
/chanserv access #channel add nick level

Retire l’acces
/chanserv access #channel del nick level

Retirer tous les acces
/chanserv access #channel clear

Liste des acces
/chanserv access #channel list

8-Topic

Changer le topic
/msg chanserv set #channel TOPIC message

Retenir le topic
/msg chanserv set #channel keeptopic on ou off

Interdire le changement
/msg chanserv set #channel topiclock on ou off

Utiliser la clef
/join #chan clef

9-Botserv

Aide
/msg botserv help

Liste des bots
/msg botserv botlist

Placer un bot
/msg botserv assign #channel nom_du_bot

Retirer un bot
/msg botserv unassign #channel nom_du_bot

Infos sur bot
/msg botserv info nick_du_bot

Autoriser fantasy sur le chan (op on le droit de kicker)
/msg botserv set #chan fantasy on ou off

Faire causer le bot
/msg botserv say #chan message

Faire kicker, deop, op, voice, devoice par le bot
/botserv kick #chan nick message (c’est le bot qui kickera comme une personne sur le chan)

Faire kicker, deop, op, voice, devoice par le bot
!kick nick message (ce message s’affichera comme une parole ecrite puis le bot executera l’ordre)

10-Mots Interdits

Ajouter un mot interdit
/msg botserv badwords #channel add mot

Retirer un mot interdit
/msg botserv badwords #channel del mot

Liste des mots
/msg botserv badwords #channel list
