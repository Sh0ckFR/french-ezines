
Le KeygenMe Hccc#5.
---------------------

Le but est simple, en faire une keygen. J'ai rajouté un petit Anti-Softice
très simple à contourner donc il faudra s'en occuper avant :). Sinon rassurez
vous, il reste assez simple à keygenner. Les infos fournient dans le zine sont
suffisantes pour en venir à bout.

- La première est de faire un KeyGenerator qui fournira donc une clé valide.
- La deuxième est un Memory Patch pour virer l'Anti-Si (voir le cours de Christal
dans le zine)

Vous pouvez même faire les 2. Ensuite il suffit de m'envoyer un tutorial
à hccc@caramail.com. Vous aurez toutes les chances d'être publié au prochain
numéro.
