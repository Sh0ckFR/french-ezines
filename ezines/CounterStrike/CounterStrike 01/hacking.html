MAILS ANNONYMES


     Pour ça il faut TELNET bon il est sous UNIX et WINDOWS. Pour WINDOWS aller dans DEMARRER puis EXECUTER puis TELNET. Quand il est ouvert (ou bleu) cliquer (en haut) sur CONNEXION puis SYSTEME DISTANT. Dans la fenêtre mettre dans NOM D'HOTE: (le server de mail) ex: mail.wanadoo.fr puis dans PORT mettre le port 25 puis pour le TYPE DE TERMINAL laisser celui qui est en place. Après faites CONNECTER. ATTENTION aller dans TEMINAL (en haut) puis cliquer sur ECHO LOCAL (sinon eh bien vous n'avez cas faire un essai sans).
Bâ maintenant va faloir faire causette avec le server:

Quand il se connecte, il vous dit:
220 le_server_de_mail_que_vous_avez_choisis SendMail X-320 Io v40.5 Ready at la_date_du_jour_et_l'heure

Maintenant va faloir lui dire Bonjour faites:
HELO le_server_de_mail_que_vous_avez_choisis (helo avec un seul L)

Il vous repond:
250 le_server_de_mail_que_vous_avez_choisis Hello votre.domain.com (votre.domain.com), pleased to meet you

Il vous demande de vous presenter donc taper un truc débile sachant qu'il ne faut pas que cela soit: xxxxx@xxxx.xxx mais plutôt: nom_à_la_con@wanadoo.fr (si vous avez pris comme server mail wanadoo) donc taper:
MAIL FROM: <l'adresse_E_mail_choisi>

Il vous repond:
250 <l'adresse_E_mail_choisi> ... Sender ok !

Maintenant mettre la personne à qui vous voulez envoyer le message taper:
RCPT TO: <votre_cible_pour_le_E_mail>

Il vous repond:
250 <votre_cible_pour_le_E_mail> ... Recipient ok !
