ETAPE
 

PROCEDURE
1
Bon, executez le programme TELNET.EXE sous le répertoire c:\windows. Si vous ne l'avez pas demerdez vous pour le recuperer.
2
Allez dans le menu PREFERENCE et cochez les cases suivantes :
Echo local (pour voir ce que vous ecrivez)
VT100/ANSI (c'est le type de terminal le plus courant)
3
Après cela, cliquez sur le menu CONNECTION, choisissez SYSTEME DISTANT et une boiboite s'ouvre.
Dans "PORT" prenez 25 (c'est le port telnet)
Dans "Type de terminal" mettez VT100.

Ensuite il faut choisir un serveur qui acceptera de reacheminer votre mail. Je vous en donne 2 en guise d'exemple qui fonctionnent, mais il y en a plein d'autre (à vous de les trouver). Si les 2 exemples ne fonctionnent plus, essayez avec d'autres serveurs.

www.thomson.com / www.apec-asso.fr
4
Ensuite vous cliquez sur CONNECTER.

Et voici maintenant un exemple concret d'envoi de mail en passant par thomson.com

Donc Une fois connecté comme expliqué plus haut, le serveur devrait vous répondre un truc du genre:

220 zelda.thomson.com ESMTP Sendmail 8.9.2/8.9.2; Thu, 1 Jul 1999 06:49:55 -0400 (EDT)

A présente il faut taper:

HELO ESMTP

(ESMTP est valable dans ce cas la mais ca dépend en fait de ce que le serveur vous répondra avant, à vous de voir et de changer les ordres en conséquence). Le serveur vous répond:

250 zelda.thomson.com Hello ppp-66-13.infonie.fr [195.242.66.13], pleased to meet you

 
5
Maintenant il va falloir choisir ce que vous voulez que le destinataire voit apparaitre dans la fenêtre "from", c'est a dire l'adresse du mec pour qui vous vous ferez passer.
MAIL FROM: <hacker@amateur.com>

Le serveur répondra alors:

250 <hacker@amateur.com>... sender ok

(Ici vous vous ferez passez pour un mec s'appelant hacker et étant abonné à amateur. C'est du pipo mais n'importe quel ISP peux faire l'affaire (ex : compuserve.com / infonie.fr / elysee.fr).
6
Là vous rentrez le nom du destinataire:
RCPT TO: <administrateur@thomson.com>

(A savoir que l'adresse doit obligatoirement exister sinon ca plante). Le serveur vous indique alors qu'il a trouvé le destinataire:

250 <administrateur@thomson.com>... Recipient ok

7
 

Vous allez maintenant pouvoir taper le message mais avant de le faire, vous devez indiquez que vous voulez taper un message en mettant:

DATA

Le serveur vous répond et vous indique comment faire:

354 Please enter your message body, ending with a "." on a line by itslef

Là vous tapez le message que vous voulez en faisant bien attention de terminer chaque phrase par un point ou un tiret. Pour terminer votre message, mettez juste un point sur la dernière ligne.

Bonjour Monsieur l'adminstrateur,
bla blablabla.
z auriez pas une tele 16/9 pour moi?.
si oui merci de l'envoyer che moi a mon adresse pipo.
bonne journee.
.

(La vous avez envoyé le message sans oublier de mettre le point sur une ligne seule)

Le serveur vous répond que le courrier est envoyé:

250 Message accepted for delivery

8
Elle est pas belle la vie? Et pour quitter du serveur vous tapez :
QUIT
