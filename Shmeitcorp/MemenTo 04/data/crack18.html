La manière esthétique de Cracker un PrG.

 

Intro :

Ici je vais vous parler de la manière esthétique de Cracker un PrG.

Par ex : Vous avez cracker la version Trial d’un PrG mais vous avez un vilain nag ou il est marqué en grand TRIAL, c’est chiant, non ?

Moi ça m’énerve !

Donc je vais vous expliquer comment effacer ça sans trop se casser les couilles …

Parce quand il faut effacer des textes de près d’une page en utilisant un éditeur hexadécimale, bonne chance !

 

Outils nécessaires :

Un éditeur de ressource comme : ExeScope, Ressource Hacker …(Disponible sur mon site ou sur celui de ShmeitCorp).

Pour commencer, utiliser ExeScope, c’est le + facile.

[ Ici nous utiliserons comme exemple StarClean ( il a tellement été crk qu’il ferait mieux d’aller dans le domaine public …) et WinAmp 2.5 (Même si c’est devenu un FreeWare, c’est quand même un bon exemple ...  et puis presque tout le monde l’a) ]

 

Bon let’s Go !

Avant Tout :

- Tout d’abord faites une sauvegarde du PrG ( Par ex : PrGdeMerde.exe PrGSav.exe )

Après lancer eXeScope et demander lui de lancer votre PrG.

- Et N’oubliez jamais de confirmer vos modifications en allant dans Menu_Edit

>SaveUpdate> Puis vous avez une fenêtre de confirmation, Cliquer OK.

- Vérifiez aussi si votre programme n’est pas en lecture seule.

- N’oubliez pas que si votre PrG est compressé de le décompresser sinon vous ne saurez pas le modifier dans ExeScope.

 

  • 1°)  Bon, imaginons qu’une Image vous agace, OK ?

Vous voulez la virer donc vous ouvrez eXeScope, vous allez voir dans ressource ( le reste, on s’en Fou !) et vous allez sûrement voir qqch comme bitmap ou JPEG ou  GIF …

Vous regardez si votre image s’y trouve sinon il faudra essayer les DLLs

Vous voulez la remplacer Donc vous cliquer là   sur le bouton 2 (Export) dans la barre d’outils plus haut.

Et  eXeScope vous propose de sauvegarder votre ImG.

Vous la modifiez avec votre logiciel de retouche d’image (Cracké évidemment ! )

Mais faite très attention à respecter le format indiquer par eXeScope en bas à gauche.  

Après Tout ça : Vous retournez dans eXeScope et cliquer Sur le bouton 1  

 

Si eXeScope vous affiche un message d’erreur, c’est que la photo n’est pas au bon format ou taille ou encore il y a trop de couleurs. (en général 256 pour un BMP)

Puis pour Sauvegardez, vous allez dans le Menu _Edit_ >SaveUpdate> Puis vous avez une fenêtre de confirmation, Cliquer OK.

Fermez eXeScope et lancez-le et visualiser le résultat !

Notez sur un papier votre phrase et lancez eXeScope( et tout le reste …) puis allez voir dans ressource > String, Dialog ou texte ( ou qqch dans le genre, ça change selon les PrG …), vous aurez un sous-menu avec des chiffres et allez y !

Et donc ces Phrases dans ExeScope : (dialog)

Bon le n°1 : Le titre, quand vous ouvrez dialog > nombre (113 par ex ) et que vous arrivez dans l’endroit général, vous verrez « caption : Start Clean » (1), vous pouvez remplacer la phrase par ce que vous voulez  (PrG de Merde par ex).

Bon le n°2 : une phrase n’importe où, donc vous cliquer sur « static : votre phrase »

Même chose, vous la modifiez comme vous voulez.

Les chiffres au dessus n’ont pas beaucoup d’importance pour le moment, mais ça veut dire où se trouve telle ou telle phrase à tel ou tel endroit.

Bon le n°3 : c’est un bouton mais c’est le même principe.

Aussi « AutoCheckBox », c’est aussi un bouton mais vous pouvez changer son apparence comme tout les autres boutons.

Bon, une dernière chose pour modifier l’aspect des fenêtres de dialog :   dans la barre d’outils plus haut.

Cliquer sur 1 et vous accédez à une fenêtre qui vous permet de modifier la place, la taille et des tas d’autres chose ! (cette Fonction se trouve uniquement dans «dialog»)

 

  • 3°) Remplacer une icône : facile ! :

  • 4°) Modifier les infos sur le programme : facile !

Vous allez dans ressource > Version > nombre (en général 1)

Vous voyez un truc comme LegalCopyright : Copyright © Programmeur_Qui_S_ennnuie

Alors vous le sélectionnez et vous faites un click droit > Edit et encore une fois vous mettez ce que vous voulez. Mais vous pouvez aussi supprimer cette ligne ou encore en insérer une nouvelle.
