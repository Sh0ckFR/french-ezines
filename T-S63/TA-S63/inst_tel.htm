Téléphonie-radiocommunications : installations téléphoniques 
 
Description 
    Par installations téléphoniques, on entend les différents types de téléphone, analogiques ou numériques, les fonctions disponibles sur chaque téléphone, les services reliés au système de messagerie vocale tels les boîtes vocales, les diffuseurs de messages simples ou à choix multiples, la réponse automatisée. 

    Parmi les appareils analogiques, on retrouve les appareils Meridian M9009. Pour les appareils numériques, la norme concerne les appareils Meridian M2006, M2008 avec ou sans afficheur, et M2616 avec ou sans afficheur et fonction "Mains libres" intégré. Des casques d'écoute sont également disponibles. 

    Pour ce qui est de la messagerie vocale, on dénombre présentement 2500 boîtes vocales sur le campus et plus de 300 diffuseurs simples ou multiples. Le système de messagerie vocale Meridian est de type 64 ports d'accès pour 240 heures d'enregistrement.

Comment s'en prévaloir 
  • Pour obtenir un diffuseur de messages : Vous devez communiquer avec nous par téléphone pour obtenir des informations pertinentes. Généralement, nous rencontrons les utilisateurs pour établir avec eux l'arborescence du diffuseur de messages en fonction de leurs besoins. Pour démarrer le projet, vous devez compléter le formulaire de demande de service (DITER-00.98). Si vous êtes familier avec les diffuseurs de messages, veuillez nous envoyer le plan de la structure de votre diffuseur. Idéalement, si vous avez déjà le texte des messages à diffuser, cela nous permet de l'évaluer plus rapidement. Lorsque les besoins sont identifiés, nous procédons à la programmation du diffuseur de messages.
  • Pour modifier un diffuseur de messages : Vous devez nous faire parvenir l'essentiel des modifications en vous servant de l'arborescence existante de votre diffuseur de messages, le tout accompagné d'un formulaire de demande de service.
  • Pour obtenir une boîte vocale : Une demande de service dûment enregistrée est requise pour l'exécution et la facturation du travail.
