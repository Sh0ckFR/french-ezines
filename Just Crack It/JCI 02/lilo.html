La lecture commence ici

Bon la je vais vous expliquer comment changer le pass root d'une machine même si celle-ci est protégée par un pass lilo.
Le seul problème et qu'il faut se trouver en face de la machine , un corp à corp. La première chose donc vous arrez besoin est un éditeur de partition style Diskedit , ( pour ceux qui ne le trouverait pas je vous ait fait une disquette contenant tout ce qui faut, il vous suffira de faire une disuqette de boot "sous dos format a: /s" et de copier tout le reste de mon fichier sur la disquette , rebootez avec et la tapez tout d'abord :

- keyb fr " pour avoir le clavier francais"
- Puis faites diskedit

La allez dans find et faite une recherche de "timeout=" par exemple.

Explication

Voici le fichier lilo.conf de ma machine contenu dans /etc/lilo.conf

boot=/dev/hda
map=/boot/map
install=/boot/boot.b
vga=normal
default=linux
keytable=/boot/fr-latin1.klt
prompt
timeout=10
password=nike
message=/boot/message
other=/dev/hda1
label=windows
table=/dev/hda
other=/dev/fd0
label=floppy
unsafe
image=/boot/vmlinuz
label=linux
root=/dev/hda7
append=""
read-only

La on peut voir que mon pass est définit password=nike , mon pass est donc nike ( c un exemple :-)). Ce fichier est présent sur mon disque dur , ( on active les options en fesant /sbin/lilo mais bon on sans fous ) Donc ce fichier présent sur mon disque contient le pass en clair, la recherche d'une des lignes permet l'affichage de ce pass ( je n'ai pas fait une recherche de "password=" car comme vous pouvez le voir j'ai aussi windows sur mon disque et lui contient plein de fois le mot "password=" alors que "timeout=" ni est pas) Si en fesant votre recherche vous tombez sur le fichier mais ne contenant pas "password=" il y à de cas , soit c parceque vous êtes dans le fichier temporaire et la une nouvelle recherche vous permettra de tomber sur le vrai fichier lilo.conf , si non c que le master à pris le soit de suprimé la ligne "password=" , dans ce cas tout n'est pas perdu ,il vous suffit de refaire une recherche de mais cette fois ci du non de systeme éventuellement bootable, dans mon fichier lilo.conf ( ci dessus) on peut voir label=windows ainsi que label=floppy et label=linux , il faut savoir que lilo garde en clair le mot écrit après label ainsi que le pass , une nouvelle recherche mais cette fois ci de windows ou linux ou autre écrit après label vous permettra de trouvé le fichier lilo compilé , la vous devriez voir écrit les différents label possible ( dans mon cas j'ai windows , floppy et linux ) ci vous n'avait pas au moins windows et linux (dans mon cas )à la suite par exemple
windows...@<...>.."quelque truc puis" linux...@<...>
c pas bon il faut continuer la recherche , dans le cas contraire vous devriez voir un mots lissible qui est tout simplement le pass de lilo.
La vous n'avez plus d'un les deux cas qu'a rebooter et mettre le pass trouver lorsque lilo vous le demande.Puis lorsqu'il vous demande avec quoi booter tapez "linux single" , linux devrait démarrer en mode minimal, la commande toute simple "passwd" pour taper un nouveau pass de votre choix, il n'y à plus maintenant qu'a rebooter tranquille
