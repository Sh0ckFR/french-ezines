Bon let's go.
Tout d'abord il existe 2 types de données :
- l'ISO 9660
- le Joliet
En mode ISO un CD pourra être lu sur tout les systèmes d'exploitation ( linux, OS/2 ...) et en mode Joliet on pourra le lire que sur Dos et Windaube : beurk !!!
Par contre en mode ISO on peux nommer le fichier avec seulement 8 caractères contre 64 pour le mode Joliet.

Les cd amorçable ou Boot :
Il faut choisir cette option si vous voulez faire démarrer vot becane avec ce CD. Il devra être graver en ISO 9660.

Les cd "extra" ou "plus":
Ce type de CD contiens 2 session dont la première supporte les pistes audios au format CD-DA et la 2eme les piste de données CD-ROM XA.
C'est cool pour faire des copies conformes de jeux ( Age of empires ...) qui contiennent des pistes audios. Par contre faut un lecteur multissession pour les lire.

Les cd mixtes
C'est la même chose mais la 1ere piste c'est des données et pis ça marche pas bien ds les lecteur de chaînes hifi !
Par contre ça tolère tous les lecteurs CD

Les cd hybrides :
Ces CD sont destinés à être lu sur un PC ( format ISO 9660 ) et un MAC ( format HFS ). Le logiciel copieras en fait toutes vos données en double. En tout cas ce genre de CD est délire et achement balèze à graver !

Les cd mode1
C'est le mode universel courant inventé par Sony ( c'est pas de la pub !), il convient pour graver une image et plus toucher au CD. Ca marche tip top et ca pete jamais.

Les cd mode2 XA
C'est le mode améliorée qui n'est pas supporté par tous les lecteurs CD ( genre le 2x de ma grand-mère) mais qui diffuse une meilleure qualité sonore et vidéo.
C'est également le mode à choisir mour composer des CD multisessions. Enfin bref gravez en mode 2 !

2 trucs :
- Ne faites pas 1 piste en mode1, la 2eme en MOde2 etc paske c'est pas bien.
- Pour aller de sessions en sessions il faut aller ds le poste de travail et clicker ( btn droit ) sur l'icône de vot lecteur CD et faire Choose session si vous avez installer Easy CD Creator ou Propriétés-Volumes si vous avez Nero 4 ( C'est pour les plus connus )
 
 LE TRUC DU MOIS DE PIGN0N :
Avez-vous déjà rencontré cette situation ? Vous connaissez le nombre de minutes qu'ils vous reste à graver mais pas la capacité en MO. Et bien voici comment faire la conversion :
le nombre de minutes * 60 ( c'est les secondes ) * 75 ( ca c(est les secteurs et y en a 75 pas secondes ) * 2 ( Ko > y en 2 par secteurs ) = taille en Ko
Pour la remettre en Mo, on sait que 1 Mo = 1024 Ko donc on fait la taille en Ko/1024
et voila on a le résultat !!
