Instruction CALL:
Appel de procédure, l'adresse de retour dans (IP) est placée sur la pile. L'adresse de 4 octets peut s'écrire sous la forme suivante : 1234h:5678h, la première valeur est le segment et la deuxième est l'offset (c'est un appel long, CS est également placé sur la pile).


Exemple : 
	#make_COM#
	ORG 100h  ; pour un fichier COM.
	call p1
	add AX, 1
	ret         ; retour au système d'exploitation.
	
	p1 PROC     ; déclaration de la procédure.
		MOV AX, 1234h
		RET     ; retour à l'appel.
	p1 ENDP
