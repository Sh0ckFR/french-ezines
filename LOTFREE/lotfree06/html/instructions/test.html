Instruction TEST:
AND (ET) logique entre tous les bits des deux opérandes positionne uniquement les flags. Les flags suivants sont affectés : ZF, SF, PF. Le résultat n'est pas utilisé.

Table de vérité :

1 AND (ET) 1 = 1
1 AND (ET) 0 = 0
0 AND (ET) 1 = 0
0 AND (ET) 0 = 0


Exemple : 
	MOV AL, 00000101b
	TEST AL, 1         ; ZF = 0.
	TEST AL, 10b       ; ZF = 1.
	RET
