Instruction SHR:
Effectue un décalage  de l'opérande1 à droite. Le nombre de décalages est contenu dans l'opérande2.

Algorithme :

    Décale tous les bits vers la droite, le bit qui déborde est placé dans CF.

    Le bit 0 à gauche est mis à 0.

Exemple : 
	MOV AL, 00000111b
	SHR AL, 1		; AL = 00000011b,  CF=1.
	
	RET
C O
r r
OF=0 si c'est le premier opérande, il conserve le signe d'origine.
