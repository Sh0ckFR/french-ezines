Instruction JNE:
Saut si la condition est réalisée.
Saut court si le premier opérande est non égal au deuxième opérande (identique à l'instruction CMP). Signé et non signé.

Algorithme :

    si ZF = 0 alors saut
Exemple : 
		include 'emu8086.inc'
		#make_COM#
		ORG 100h
		MOV AL, 2
		CMP AL, 3
		JNE  label1
		PRINT 'AL = 3.'
		JMP exit
	label1:
		PRINT 'Al <;>; 3.'
	exit:
		RET
